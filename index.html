Personal Projects:

Wellness Tracker

This wellness tracker allows the user to log sleep, nutrition, and exercise habits. Users can log in with their Facebook or Github profile. Built using React (with Redux), Express, and PostgreSQL, I chose to build this app in order to solidify my knowledge of Javascript and React, while still introducing new learned concepts.

Challenges:

  • Managing the size of a relatively larger application.
  • Properly using API calls, for both the Facebook and Github OAuth logins and the Edamam food database API.
  • Implementing Redux, as well as react-router and redux-thunk, in a larger application.

Nota Bene

Built using React, Express, and PostgreSQL, Nota Bene was my first attempt at creating a full-stack web application from scratch. The app allows a user to create and store notes, in a manner similar to the iPhone's Notes app. Users can log in at any time to access their notes.

Challenges:

  • Tying my front-end and back-end knowledge together into one cohesive app
  • Managing my application's state in React to allow for seamless creation and deletion of notes, as well as maintaining an order based on last edit time
  • Keeping the notes data on the front-end consistent with the notes data in the database
